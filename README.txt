نسخه اصلاح‌شده TAK UI

مشکل نسخه قبل:
اسکریپت PWA داخل یکی از متن‌های JavaScript تزریق شده بود و باعث شد کدها به شکل متن روی صفحه دیده شوند.

روش استفاده:
1) همه فایل‌های این پوشه را جایگزین فایل‌های قبلی GitHub کن.
2) Commit بزن.
3) در گوشی یک بار کش سایت را پاک کن یا آدرس را با ?v=2 باز کن.
مثال:
https://USERNAME.github.io/REPO/?v=2

این نسخه فقط ظاهر را تغییر داده و منطق محاسبات دست نخورده است.
